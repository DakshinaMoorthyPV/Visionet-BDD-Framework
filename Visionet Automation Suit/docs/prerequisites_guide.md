    <h3>System Variable Setup for Java Project</h3>
    <h4>Windows</h4>
    <ul>
        <li>
            <strong>Java Development Kit (JDK)</strong><br>
            <em>Variable Name:</em> JAVA_HOME<br>
            <em>Variable Value:</em> C:\Program Files\Java\jdk-17
        </li>
        <li>
            <strong>Maven</strong><br>
            <em>Variable Name:</em> MAVEN_HOME<br>
            <em>Variable Value:</em> C:\Program Files\Apache\maven-3.9.4<br>
            Add %MAVEN_HOME%\bin to the PATH variable.
        </li>
        <li>
            <strong>Appium</strong><br>
            <em>Variable Name:</em> APPIUM_HOME<br>
            <em>Variable Value:</em> C:\Program Files\Appium<br>
            Add %APPIUM_HOME%\bin to the PATH variable.
        </li>
        <li>
            <strong>Node.js</strong><br>
            Add the Node.js installation path to the PATH variable, e.g., C:\Program Files\nodejs
        </li>
        <li>
            <strong>Android SDK</strong><br>
            <em>Variable Name:</em> ANDROID_HOME<br>
            <em>Variable Value:</em> Path to your Android SDK, e.g., C:\Users\<username>\AppData\Local\Android\Sdk<br>
            Add %ANDROID_HOME%\tools and %ANDROID_HOME%\platform-tools to the PATH variable.
        </li>
    </ul>
    <h4>macOS</h4>
    <ul>
        <li>
            <strong>Java Development Kit (JDK)</strong><br>
            Add the following to ~/.bash_profile or ~/.zshrc:<br>
            <code>export JAVA_HOME=$(/usr/libexec/java_home -v 17)</code>
        </li>
        <li>
            <strong>Maven</strong><br>
            Add the following to ~/.bash_profile or ~/.zshrc:<br>
            <code>export MAVEN_HOME=/usr/local/Cellar/maven/3.9.4/libexec</code><br>
            <code>export PATH=$MAVEN_HOME/bin:$PATH</code>
        </li>
        <li>
            <strong>Appium</strong><br>
            Install Appium using npm:<br>
            <code>npm install -g appium</code>
        </li>
        <li>
            <strong>Node.js</strong><br>
            Install Node.js using a package manager like brew:<br>
            <code>brew install node</code>
        </li>
        <li>
            <strong>Android SDK</strong><br>
            Add the following to ~/.bash_profile or ~/.zshrc:<br>
            <code>export ANDROID_HOME=$HOME/Library/Android/sdk</code><br>
            <code>export PATH=$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools:$PATH</code>
        </li>
    </ul>
